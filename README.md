# xuemeigu.github.io
homepage

## Source layout

- `content/pages/` contains the main website pages.
- `content/data/` contains profile, navigation, news, publications, talks, and other list data.
- `content/collections/` contains posts and projects.
- `_includes/` and `_layouts/` contain reusable Jekyll templates.
- `assets/` contains CSS, JavaScript, images, and downloadable files.
- `docs/` contains maintenance notes that are excluded from the published site.
