# Placeholder checklist

Search the whole folder for `PLACEHOLDER` and replace items gradually.

## Essential before publishing

- `Mei YourSurname`
- `yourusername`
- `your.email@example.com`
- `PLACEHOLDER: Your University / Institute`
- `PLACEHOLDER: City, Country`
- Google Scholar, ORCID, GitHub, LinkedIn links
- Homepage research summary
- Contact page

## Recommended soon after publishing

- Real profile photo
- Real CV PDF
- Real publication list
- Real project pages
- Real teaching information
- 2–3 short notes or news items

## Optional improvements

- Add custom domain
- Add a downloadable publication list
- Add selected figures to project pages
- Add a `/group/` page if you later lead students or collaborators
- Add a `/funding/` page for grants and proposals
